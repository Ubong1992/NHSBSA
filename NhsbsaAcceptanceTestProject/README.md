# NHSBSA Acceptance Test Project

This is an automated UI test suite for the NHS Jobs website using:

- Java 21
- Selenium 4.28.1
- Cucumber
- JUnit 4.13.2
- Maven 3.9.10
- WebDriverManager
- PicoContainer for dependency injection
- Browser can be run in Chrome and Firefox

# Setup

1. Install Java 21
2. Clone the repository
3. Run:
   mvn clean install
   mvn clean verify
   mvn clean test
   mvn  test
